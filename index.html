Over Mij

Leuk dat je op mijn website terecht bent gekomen! Ik ben Britt, gediplomeerd nagelstyliste. Inmiddels 4 jaar werkzaam in deze mooie branche, waarvan dit mijn 3e jaar wordt als zelfstandig ondernemer. De salon is gevestigd in hartje centrum van Wageningen.

Mijn doel is het maken van jouw droomnagels, dit kan doormiddel van verlenging, nail art, de nagels in de perfecte vorm vijlen of wat je maar wenst. Door de juiste technieken toe te passen zorg ik ervoor dat je lang van je nagels kan genieten! Wie wilt dit nou niet?

Dit kan ik doen omdat ik verschillende cursussen heb gevolgd die mij geleerd hebben hoe ik deze technieken toe moet passen. Benieuwd welke cursussen ik heb gevolgd? Hieronder vind je een overzicht.
